# Flutter-Apps

A collection of small Flutter applications demonstrating fundamental concepts of Flutter development, including widgets, state management, navigation, and basic CRUD operations. These projects are ideal for beginners to intermediate Flutter developers who want hands-on practice.

---

## Projects Included

| App Name         | Description |
|-----------------|-------------|
| **basic_widgets_app** | Demonstrates the use of core Flutter widgets such as Text, Container, Row, Column, and Buttons. |
| **counter_app**       | Simple counter app showcasing state management with `setState`. |
| **hello_flutter**     | Basic starter app with a simple "Hello World" example in Flutter. |
| **navigation_app**    | Demonstrates navigation between multiple screens/routes in Flutter. |
| **todo_simle**        | Simple TODO list app to practice state management and UI updates. |

---

## Features

- Built entirely with **Flutter & Dart**
- Demonstrates core Flutter concepts:
  - Widgets and Layouts
  - State Management (`setState`)
  - Navigation and Routing
  - ListView & Basic CRUD functionality
- Ready to run on both **Android** and **iOS**
